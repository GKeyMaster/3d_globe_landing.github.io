# Building Textures (CC0)

**Place these 3 files here** (bundled via Vite, not loaded from /public):

- `facade_01.webp` – Facade texture (concrete tiles, wall cladding)
- `facade_02.webp` – Alternate facade texture
- `roof_01.webp` – Roof tiles texture

**Source:** Poly Haven or ambientCG (CC0). Export as WEBP at 1024px or 2048px (color/albedo only).
